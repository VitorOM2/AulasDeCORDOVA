# Como criar um projeto e enviar para o github
1 - Criar repositório
2 - Clonar repositório
3 - Entrar na pasta ( cd nomeDaPasta )
4 - Abrir o arquivo ( nomeDoArquivo )
5 - Salvar
6 - verificar o status ( git status )
7 - comite ( git commit )
8 - push ( git push )
